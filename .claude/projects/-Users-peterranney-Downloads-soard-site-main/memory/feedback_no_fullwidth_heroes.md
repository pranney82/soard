Do not use full-width/full-viewport hero sections with background images that span edge-to-edge.

**Why:** User explicitly stated "i hate full width heros" when one was used on the Sunny & Ranney page.

**How to apply:** When designing page heroes, use contained editorial layouts (e.g. split hero with text + photo inside a max-width container) instead of viewport-spanning background images. The homepage's split hero pattern (text left, photo right inside `--max-width`) is the preferred approach.
